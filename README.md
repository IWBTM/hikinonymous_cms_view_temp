# Hikinonymous front

## 프로그램 실행 방법
### CLI
1. yarn install
2. yarn start

### 저 위에 인텔리제이 실행
1. Edit Configurations 클릭
2. npm으로 잡아야함
3. scripts: start
4. Node Option: --openssl-legacy-provider 

헤더 구조
1. Search (보류)
2. Notification (생성, 삭제 등 매니저들의 이벤트 발생 시) (슈퍼 관리자)
3. Logs (본인의 로그 기록)
4. Profile (이름 + 사진)
