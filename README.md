# Hikinonymous front

## 프로그램 실행 방법
### CLI
1. yarn install
2. yarn start

### 저 위에 인텔리제이 실행
1. Edit Configurations 클릭
2. npm으로 잡아야함
3. scripts: start
4. Node Option: --openssl-legacy-provider

1. MAIN: Dashboard (금일/금월/금년 사이트 접속 수, 금일/금월/금년 게시글 작성 수, 금일/금월/금년 댓글 작성 수, 금일/금월/금년 광고 수익비)
2. 관리자 관리 (슈퍼 관리자 CRUD)

4. 회원 관리 (일반 관리자 RU)

4 - 2. 탈퇴 회원 관리
